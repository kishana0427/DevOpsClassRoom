## Git

### Installation
* Windows:
    
    * Install chocolatey from [here](https://chocolatey.org/docs/installation) 
    * Install Git For Windows
    ```
    choco install git -y
    ```
* Linux:
    * ubuntu: 
    ```
    sudo apt-get install git -y
    ```
    * redhat
    ```
    sudo yum install git -y
    ```
