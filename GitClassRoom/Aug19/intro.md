# GIT

## What is it?
1. Distributed Version Control System


## How to Install It?
### Windows
1. Install Chocolatey
[Refer](https://chocolatey.org/docs/installation)

2. Install Git
```
choco install git -y
```

### Linux

#### Ubuntu
```
sudo apt-get update && sudo apt-get install git -y
```

#### Redhat
```
sudo yum install git -y
```
